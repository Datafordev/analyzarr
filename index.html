Getting Analyzarr up and running

All platforms

Install from source!

  1. Clone the git repository - https://github.com/msarahan/analyzarr
  2. Run setup.py - preferably with pip:
pip setup.py install

or

python setup.py install

Windows x64 shortcut

A customized installation of WinPython () has been created that has all of Analyzarr's prerequisites installed. You can download it from here:

http://t3hmikez0r.com/static/analyzarr/WinPython-64bit-2.7.3.3.7z

The Analyzarr installation in that folder is not kept up to date. To get a more up-to-date version, you have two options:

  1. Download a binary installer for Analyzarr. These are built infrequently. They are here: http://t3hmikez0r.com/static/analyzarr/

  2. Build a new binary installer. Clone the source code from github, then go to that folder and run this command:

python setup.py bdist_wininst

Either way, install the binary installer like so:

  1. Install this by opening the WinPython control panel.
  2. Uninstall the previously installed version of analyzarr
  3. Drag and drop the binary installer into the WinPython control panel's install tab.
